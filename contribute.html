--- layout: page title: Contribute optional-title: How to contribute to gellius-db permalink: /contribute/ --- Do you think a commentarius lacks important information? Are you missing a certain feature on the website? Or have you spotted an error in a query?
There are several things you can do to improve gellius-db:
  1. Feel free to fork this site's repository at GitHub, add your own features and send a pull request when you are satisfied with your results.
  2. Improve the data gellius-db is working with. You can either edit the data at Wikidata yourself or you can fill in a form which adds your enhancements to the respective data set automatically. (How does this work?)
  3. Write an e-mail to gellius-db[at]outlook.com.
